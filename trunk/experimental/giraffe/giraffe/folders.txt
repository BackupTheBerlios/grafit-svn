Nothing
-------

The following special names are defined:

* `here`: the current folder

* `up`: the parent of the current folder

* `top`: the top-level folder

    >>> from giraffe import Project, Worksheet
    >>> from giraffe.common.commands import undo, redo
    >>> p = Project()
    >>> p.set_dict(globals())
    >>> p.mkfolder('ass')
    >>> ass
    <Folder ass>
    >>> p.rmfolder('ass')
    >>> ass
    Traceback (most recent call last):
    ...
    NameError: name 'ass' is not defined
    >>> p.rmfolder('hole')
    Traceback (most recent call last):
    NameError: folder 'hole' does not exist

    >>> w = p.new(Worksheet, 'data1')
    >>> data1 == w
    True

    >>> undo()
    >>> data1 == w
    Traceback (most recent call last):
    ...
    NameError: name 'data1' is not defined

    >>> redo()
    >>> data1 == w
    True

    >>> p.remove(data1.id)
    >>> data1 == w
    Traceback (most recent call last):
    ...
    NameError: name 'data1' is not defined

    >>> undo()
    >>> data1 == w
    True
