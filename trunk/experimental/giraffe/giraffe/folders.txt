* `here`: the current folder

* `up`: the parent of the current folder

* `top`: the top-level folder

    >>> from giraffe.common.commands import command_list
    >>> p = Project()
    >>> p.set_dict(globals())
    >>> p.mkfolder('ass')
    >>> ass
    <Folder ass>
    >>> p.rmfolder('ass')
    >>> ass
